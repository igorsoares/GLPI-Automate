# GLPI-Automate
Automation script to installation of GLPI.

# What he does?

* Install / Download GLPI 9.5.1;
* Download all needed packets (PHP,APACHE, MARIADB...);
* Configure database (MariaDB10) and users.
  * Default user-access : glpi(username) / glpi(password)
